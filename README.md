# 📊 Marvin's Reconciliation Assistant

This mini project was built to improve the daily reconciliation process at work — helping to spot discrepancies faster and close out with greater accuracy using Excel.

---

## 🧠 Project Objective

The goal of this tool was to streamline and automate part of the end-of-day reconciliation process. It was designed to reduce manual errors, highlight mismatches, and save time spent cross-checking balances.

---

## 🛠️ Tools Used

- **Microsoft Excel**
  - `SUM` and `MULTIPLICATION` formulas
  - Conditional formatting
  - Simple logic-based automation

---

## ⚙️ Key Features

- 🔍 **Discrepancy Highlighting** – Automatically identifies mismatches in values
- ⏱️ **Faster Reconciliation** – Reduces manual effort and speeds up close-of-business tasks
- ✅ **Improved Accuracy** – Minimizes human error during balancing

---

## 📈 Outcome

This tool helped reduce reconciliation time and made the process more consistent and error-free, especially in a fast-paced teller environment.
